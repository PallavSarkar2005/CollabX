# CollabX

CollabX is a real-time collaborative coding platform where multiple users can join a room, edit code simultaneously, chat, and see updates instantly.

## Tech Stack
- Next.js
- Socket.io
- Node.js
- Prisma
- PostgreSQL
- Monaco Editor
- TailwindCSS
- Framer Motion

## Features
- Real-time collaborative editor
- Multi-user rooms
- Live chat
- Auto-save with database
- Modern animated UI
